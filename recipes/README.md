\# Odin Recipes - Argentinian Cuisine



A practice project demonstrating HTML skills by building a recipe website focused on traditional Argentinian cuisine. This project showcases proper HTML structure, semantic elements, and multi-page navigation.



\## Project Features

\- Three authentic Argentinian recipes (Beef Empanadas, Dulce de Leche, Maizena Alfajores)

\- Semantic HTML5 structure

\- Proper document hierarchy with headings

\- Image implementation with alt text

\- Navigation between pages

\- Organized file structure



\## Skills Demonstrated

\- HTML5 semantic elements

\- Multi-page website structure

\- Lists (ordered and unordered)

\- Links and navigation

\- Image optimization

\- File organization

\- Git version control

\- Clean, readable code structure



\## Technologies Used

\- HTML5

\- Git \& GitHub

\- Markdown (for documentation)
